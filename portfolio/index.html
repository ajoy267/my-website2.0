My Portfolio

Unplugged Retreat

The Unplugged Retreat project that I worked on was a fully responsive multi-page website built from scratch using HTML, CSS, and Photoshop. This project was really fun to work on and did incorporate some challenges that were good to learn off of.

Magic 8 Ball

The Magic 8 Ball that I worked on is a fully digitial version that incorporates the use of HTML, CSS, and Javascript to make it functional! The Magic 8 Ball prompts you to ask a yes or no question, then gives you a shaking animation, followed by a response that a traditional Magic 8 ball would give.

Rogue Pickings

The Rogue Pickings project was a static single page resturaunt menu that I made from HTML, CSS, and Photoshop, it's a bit simple but like the Unplugged Retreat project I built it from scratch using the same coding language.
