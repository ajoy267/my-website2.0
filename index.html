My Portfolio

Here is a brief overview of some of the projects I have worked on!

Unplugged Retreat

The Unplugged Retreat project that I worked on is a fully responsive multi-page website built from scratch using HTML, CSS, and Photoshop. This project was really fun to work on and incorporated some challenges that were good to learn off of.

Magic 8-Ball

The Magic 8 Ball that I worked on is a fully digitial version that incorporates the use of HTML, CSS, and Javascript to make it functional! The Magic 8 Ball prompts you to ask a yes or no question, then gives you a shaking animation, followed by a response that a traditional Magic 8 ball would give.

Rogue Pickings

The Rogue Pickings project is a responsive single page resturaunt menu that I made from HTML, CSS, and Photoshop, like the Unplugged Retreat project I built it from scratch using the same coding language.

About Me

Hi, I’m Andrew Joy. After getting my start in mechanical engineering and kinesiology, I found myself gravitating more to web development. You could say my interest in mechanics went from machines to the body to code. Learning new coding languages and seeing how everything works behind the scenes is pretty exciting. Coming from an athletic and competitive background, I like taking on challenges and the feeling of accomplishment when achieving a goal. Everyone has goals and inspirations that motivate them to work towards accomplishing the things that they seek out to do. What motivates me is doing something that I believe makes an impact in the world. Knowing that I tried my best to accomplish the goals that I set out for myself, no matter how big or small the goals are is another huge motivator for me. Outside of coding, I’m a triathlete and cyclist who loves to travel.
